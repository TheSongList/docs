API

This is the Robloxian API for now it requires a token to be used.
The token can be directly requested from me on discord TheSongList#0001

Base UR

  • Main URL - https://api.songlist.moe/api/token/

  • Token - The API token

Images

These are the image requests!

cat

This will deliver a random cat image url.

REQUEST

GET

/cat

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
"cat": "https://cdn.songlist.moe/images/cat/BaETngD.jpeg"
}

Generators

These allow you to make custom images & gifs!

trump

This will create a trump tweet image. Just replace the /text/ parameter with any text!

REQUEST

GET

/trump/{text}
Parameters
text string

The text you want to appear in the tweet.

RESPONSE

Headers
Content-Type image/png
200
BODY SCHEMA

triggered

This will create a triggered gif. You have to send a header “avatar” with the image url.

REQUEST

GET

/triggered

RESPONSE

Headers
Content-Type image/png
avatar the url of the avatar
200
BODY SCHEMA
